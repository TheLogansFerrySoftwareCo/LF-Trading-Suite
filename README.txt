READ ME for LF Trading Suite

Logans Ferry (LF) Trading Suite is a collection of tools for developing trading strategies for stocks/commodities.  The suite is still in its infancy, so don't expect much yet, and contributions will always be appreciated!

LICENSE

This project is licensed under the GPL 3.0.

SUBPROJECTS

1) Database Console:  An application for managing the suite's database containing stock price history.  This app will allow the user to import ticker symbols into the database, and to retrieve price history for ticker symbols from Google.com.

2) LF Back Tester Prototype:  An initial prototype for backtesting trading strategies using the price histories from the databse.
